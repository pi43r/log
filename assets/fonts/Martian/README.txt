Prefer variable `ttf` or static `otf` font files for macOS, and `ttf` for Windows. Use `woff2` and `woff` for web development.

---

The font has 4 styles on the width axis:
	Wd  — Wide
	Std — Standard
	Nr  — Narrow
	Cn  — Condensed

And 7 weights:
	Th	— Thin
	xLt	— Extra Light
	Lt	— Light
	Rg	— Regular
	Md	— Medium
	Bd	— Bold
	xBd	— Extra Bold

---

The font contains hinting, but for the best results use the next pair of size / line height:

7.5 / 10 (or 14, 18) px
10 / 12 (or 16, 20) px
12.5 / 14 (or 18, 22) px
15 / 20 (or 24, 28) px
etc.
